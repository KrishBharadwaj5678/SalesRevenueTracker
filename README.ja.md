<h1 align="center">📊 Sales & Revenue Tracker</h1>

<p align="center">
  2001年から2024年までの企業の売上および収益データを追跡・分析・可視化するために設計された、StreamlitベースのインタラクティブなWebアプリケーションです。直感的なダッシュボードと分かりやすいビジュアライゼーションを提供し、データの傾向を把握してデータに基づいた意思決定を支援します。
</p>

## <img src="https://github.com/KrishBharadwaj5678/SalesRevenueTracker/raw/main/assets/readme/features.gif" width="35"/> 主な機能

| 機能 | 説明 |
|------|------|
| 📈 売上分析 | 2001年～2024年までの売上データを分析 |
| 💰 収益分析 | 収益の推移やパフォーマンスを可視化 |
| 📊 インタラクティブUI | Streamlitによる使いやすいユーザーインターフェース |
| ⚡ リアルタイム更新 | 入力内容に応じて結果を即座に更新 |
| 📅 時系列分析 | 年ごとのデータを詳細に分析 |
| 🌐 Webベース | ブラウザからどこでも利用可能 |

## <img src="https://github.com/KrishBharadwaj5678/SalesRevenueTracker/raw/main/assets/readme/gettingStarted.gif" width="35"/> はじめに

### 1️⃣ リポジトリをクローンする

```bash
git clone https://github.com/KrishBharadwaj5678/SalesRevenueTracker.git
```

### 2️⃣ プロジェクトディレクトリへ移動する

```bash
cd SalesRevenueTracker
```

### 3️⃣ 必要な依存関係をインストールする

```bash
pip install -r requirements.txt
```

### 4️⃣ アプリケーションを起動する

```bash
streamlit run app.py
```
