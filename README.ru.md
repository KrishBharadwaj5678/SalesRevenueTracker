<h1 align="center">📊 Sales & Revenue Tracker</h1>

<p align="center">
  Интерактивное веб-приложение на базе Streamlit, предназначенное для отслеживания, анализа и визуализации показателей продаж и доходов компании за период с 2001 по 2024 год. Приложение предоставляет удобные панели управления и информативные визуализации, помогая выявлять тенденции и принимать обоснованные решения на основе данных.
</p>

| ------------ | -------- |

## <img src="https://github.com/KrishBharadwaj5678/SalesRevenueTracker/raw/main/assets/readme/gettingStarted.gif" width="35"/> Начало работы

### 1️⃣ Клонируйте репозиторий

```bash
git clone https://github.com/KrishBharadwaj5678/SalesRevenueTracker.git
```

### 2️⃣ Перейдите в каталог проекта

```bash
cd SalesRevenueTracker
```

### 3️⃣ Установите зависимости

```bash
pip install -r requirements.txt
```

### 4️⃣ Запустите приложение

```bash
streamlit run app.py
```
