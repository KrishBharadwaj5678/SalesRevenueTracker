<h1 align="center">📊 Sales & Revenue Tracker</h1>

<p align="center">
  Um aplicativo web interativo desenvolvido com Streamlit para acompanhar, analisar e visualizar o desempenho de vendas e receitas de uma empresa entre 2001 e 2024. Ele oferece painéis intuitivos e visualizações informativas que ajudam os usuários a identificar tendências e tomar decisões baseadas em dados.
</p>

## <img src="https://github.com/KrishBharadwaj5678/SalesRevenueTracker/raw/main/assets/readme/features.gif" width="35"/> Funcionalidades

| Funcionalidade | Descrição |
| -------------- | --------- |
| 📈 Acompanhamento de Vendas | Analise os dados de vendas entre 2001 e 2024 |
| 💰 Análise de Receitas | Visualize tendências e desempenho das receitas |
| 📊 Interface Interativa | Interface amigável desenvolvida com Streamlit |
| ⚡ Atualizações em Tempo Real | Os resultados são atualizados instantaneamente com base nas entradas |
| 📅 Análise Temporal | Explore os dados de diferentes anos |
| 🌐 Baseado na Web | Acessível a partir de qualquer navegador |

## <img src="https://github.com/KrishBharadwaj5678/SalesRevenueTracker/raw/main/assets/readme/gettingStarted.gif" width="35"/> Como Começar

### 1️⃣ Clone o Repositório

```bash
git clone https://github.com/KrishBharadwaj5678/SalesRevenueTracker.git
```

### 2️⃣ Acesse o Diretório do Projeto

```bash
cd SalesRevenueTracker
```

### 3️⃣ Instale as Dependências

```bash
pip install -r requirements.txt
```

### 4️⃣ Execute a Aplicação

```bash

```
